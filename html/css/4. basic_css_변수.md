# CSS 변수(Custom Properties) 학습 교안

## 학습 목표
- CSS 변수의 개념과 필요성을 이해한다
- CSS 변수를 선언하고 사용하는 문법을 익힌다
- 변수의 스코프(적용 범위)와 상속을 이해한다
- `var()` 함수의 fallback 값을 활용할 수 있다
- 반응형 디자인과 다크모드 등 실전 예제에 CSS 변수를 적용할 수 있다
- JavaScript로 CSS 변수를 동적으로 제어할 수 있다

---

## 1차시. CSS 변수란 무엇인가?

### 1-1. 개념 소개

CSS 변수(공식 명칭: **CSS Custom Properties**)는 반복적으로 사용하는 값(색상, 간격, 폰트 크기 등)을 하나의 이름으로 저장해두고, 필요할 때마다 재사용할 수 있게 해주는 기능입니다.

**변수를 쓰지 않을 때의 문제점:**

```css
.header {
  color: #3498db;
  border: 1px solid #3498db;
}
.button {
  background-color: #3498db;
}
.link {
  color: #3498db;
}
```

위 코드에서 `#3498db`라는 색상값이 4곳에 흩어져 있습니다. 만약 브랜드 컬러가 바뀐다면 모든 곳을 일일이 찾아 수정해야 합니다.

**변수를 사용할 때:**

```css
:root {
  --main-color: #3498db;
}
.header {
  color: var(--main-color);
  border: 1px solid var(--main-color);
}
.button {
  background-color: var(--main-color);
}
.link {
  color: var(--main-color);
}
```

이제 `--main-color` 값 하나만 바꾸면 모든 곳에 자동으로 반영됩니다.

### 1-2. CSS 변수의 장점

| 장점 | 설명 |
|---|---|
| 유지보수 용이 | 한 곳에서 값을 수정하면 전체에 반영 |
| 일관성 유지 | 디자인 시스템(색상, 간격 등) 통일 |
| 동적 제어 가능 | JavaScript로 실시간 변경 가능 (Sass 변수는 불가능) |
| 스코프 활용 | 특정 요소 안에서만 다른 값 적용 가능 |
| 다크모드 구현 | 테마 전환에 매우 유용 |

> **참고:** Sass, Less 같은 전처리기의 변수는 컴파일 시점에 값이 고정되지만, CSS 변수는 브라우저가 렌더링하는 동안 실시간으로 값을 바꿀 수 있습니다.

## 2차시. 기본 문법

### 2-1. 변수 선언하기

CSS 변수 이름은 반드시 `--`(하이픈 2개)로 시작합니다.

```css
:root {
  --primary-color: #ff6347;
  --spacing-unit: 8px;
  --font-main: 'Noto Sans KR', sans-serif;
}
```

`:root`는 문서 최상위(HTML 요소)를 가리키는 가상 클래스로, 전역(global)에서 변수를 사용하고 싶을 때 관례적으로 사용합니다.

### 2-2. 변수 사용하기: `var()` 함수

```css
.box {
  background-color: var(--primary-color);
  padding: var(--spacing-unit);
  font-family: var(--font-main);
}
```

### 2-3. Fallback(대체) 값 지정하기

변수가 정의되지 않았거나 유효하지 않을 경우 사용할 기본값을 지정할 수 있습니다.

```css
.box {
  /* --undefined-color가 없으면 검정색(black) 사용 */
  color: var(--undefined-color, black);
}
```

Fallback 값에 다른 변수를 연쇄적으로 넣을 수도 있습니다.

```css
color: var(--text-color, var(--default-color, #333));
```

## 3차시. 스코프(Scope)와 상속

### 3-1. 전역 변수 vs 지역 변수

CSS 변수는 **선언된 요소와 그 하위 요소에서만** 유효합니다. 즉 일반 CSS 상속 규칙을 따릅니다.

```css
:root {
  --box-color: skyblue; /* 전역 변수 */
}

.card {
  --box-color: tomato; /* .card 내부에서만 재정의(지역 변수) */
}

.box {
  background-color: var(--box-color);
}
```

```html
<div class="box">하늘색 배경</div>

<div class="card">
  <div class="box">토마토색 배경</div>
</div>
```

`.card` 안의 `.box`는 가장 가까운 조상에서 재정의된 `tomato` 값을 사용하게 됩니다. 이것이 CSS 변수의 **캐스케이딩(cascading)** 특성입니다.

### 3-2. 미디어 쿼리와 함께 사용하기

```css
:root {
  --font-size-base: 16px;
}

@media (max-width: 768px) {
  :root {
    --font-size-base: 14px;
  }
}

body {
  font-size: var(--font-size-base);
}
```

미디어 쿼리 조건에 따라 변수 값 자체를 바꾸면, 이를 사용하는 모든 곳에 자동으로 반응형이 적용됩니다.

## 4차시. 실전 활용 예제

### 4-1. 다크 모드 구현하기

```css
:root {
  --bg-color: #ffffff;
  --text-color: #1a1a1a;
}

[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #f5f5f5;
}

body {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: background-color 0.3s, color 0.3s;
}
```

```html
<html data-theme="dark">
```

`data-theme` 속성 값만 바꿔주면 전체 테마가 전환됩니다.

### 4-2. 디자인 시스템(스페이싱, 색상 팔레트) 구축

```css
:root {
  /* 색상 팔레트 */
  --color-primary: #4361ee;
  --color-secondary: #3f37c9;
  --color-danger: #f72585;
  --color-gray-100: #f8f9fa;
  --color-gray-900: #212529;

  /* 스페이싱 단위 */
  --space-xs: 4px;
  --space-sm: 8px;
  --space-md: 16px;
  --space-lg: 32px;
  --space-xl: 64px;

  /* 폰트 크기 */
  --fs-sm: 0.875rem;
  --fs-base: 1rem;
  --fs-lg: 1.25rem;
  --fs-xl: 2rem;
}
```

## 5차시. JavaScript와 연동하기

### 5-1. JS로 CSS 변수 읽기

```javascript
const rootStyles = getComputedStyle(document.documentElement);
const primaryColor = rootStyles.getPropertyValue('--primary-color');
console.log(primaryColor); // ' #3498db' (앞에 공백 포함될 수 있음)
```

### 5-2. JS로 CSS 변수 변경하기

```javascript
document.documentElement.style.setProperty('--primary-color', '#e74c3c');
```

### 5-3. 실전 예제: 버튼 클릭으로 테마 색상 변경하기

```html
<button onclick="changeColor()">색상 변경</button>

<script>
function changeColor() {
  const randomColor = '#' + Math.floor(Math.random()*16777215).toString(16);
  document.documentElement.style.setProperty('--primary-color', randomColor);
}
</script>
```

## 6차시. 정리 및 주의사항

### 6-1. 핵심 요약

1. CSS 변수는 `--변수명: 값;` 형태로 선언한다.
2. `var(--변수명, 기본값)`으로 사용한다.
3. `:root`에 선언하면 전역에서 사용 가능하다.
4. 특정 요소 안에서 재정의하면 그 하위에서만 다르게 적용된다(캐스케이딩).
5. JavaScript로 실시간 제어가 가능하다는 점이 Sass 변수와의 가장 큰 차이다.

### 6-2. 주의사항

- 변수명은 대소문자를 구분합니다. (`--Main-Color`와 `--main-color`는 다른 변수)
- 변수 값은 반드시 유효한 CSS 값이어야 하며, 잘못된 값을 넣으면 무시되고 fallback이나 상속값이 사용됩니다.
- `var()`는 속성값 전체가 아니라 값의 일부로도 사용할 수 있습니다. 단, 선택자(selector)나 속성 이름 자체에는 사용할 수 없습니다.

```css
/* 가능 */
.box { border: 1px solid var(--color); }

/* 불가능 (속성 이름에는 var 사용 불가) */
.box { var(--prop-name): red; } /* 오류 */
```
