스타일 가이드

basic.css 사용 가이드

너비, 높이, flex, 여백처럼 매 프로젝트마다 반복해서 쓰는 스타일을 클래스 하나로 바로 쓸 수 있게 모아둔 유틸리티 스타일시트입니다. 이 페이지의 모든 예시는 실제 basic.css 클래스를 그대로 사용해 렌더링됩니다.

<div class="d-flex items-center
  justify-between gap-3">
  ...
</div>
01

사용법

파일 하나만 프로젝트 폴더에 넣고 링크하면 끝

basic.css 파일을 프로젝트 폴더에 넣고, HTML의 <head>에 아래 한 줄을 추가하세요.

<head> <link rel="stylesheet" href="basic.css"> </head>

그 다음부터는 클래스 이름만 조합해서 스타일을 바로 적용할 수 있습니다.

<div class="d-flex justify-between items-center p-4 bg-white rounded shadow"> <span class="fw-bold fs-lg">제목</span> <button class="bg-primary text-white rounded-full px-4 py-2">버튼</button> </div>
